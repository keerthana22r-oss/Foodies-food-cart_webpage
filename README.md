# 🍕 Foodies - Food Cart Webpage

A responsive food cart web application where users can browse food items, manage their cart, and place orders with a smooth login/signup experience.

---

## 🌐 Live Demo
> Coming soon via GitHub Pages

---

## ✨ Features

- 🏠 **Homepage** — Attractive landing page showcasing the Foodies brand
- 🍔 **Food Menu** — Browse a variety of food items with details and pricing
- 🛒 **Shopping Cart** — Add, remove, and manage food items in the cart
- 🔐 **Login & Signup** — User authentication pages for account access

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Page structure |
| CSS3 | Styling & layout |
| JavaScript | Interactivity & cart logic |

   ```bash
   git clone https://github.com/keerthana22r-oss/Foodies-food-cart_webpage.git
   ```

   ```
   Open homepage.html in your browser
   ```
